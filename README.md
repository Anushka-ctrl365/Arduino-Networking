# Arduino Networking

## Overview

This repository contains a series of experiments performed to study Ethernet-based communication between Arduino Uno boards using Ethernet Shields and W5500 Ethernet Modules.

The objective of the project was to understand SPI communication, TCP/IP networking, Ethernet communication, and the implementation of network topologies using embedded systems.

---

## Hardware Used

* Arduino Uno Boards
* Ethernet Shields
* W5500 Ethernet Modules
* Router
* LAN Cables
* Breadboard
* Push Buttons
* LEDs
* Jumper Wires

---

## Experiments

### Experiment 1

Communication between two Arduino Uno boards using Ethernet Shields.

### Experiment 2

Communication between an Ethernet Shield and a W5500 Ethernet Module.

### Experiment 3

Communication between two W5500 Ethernet Modules through a router.

### Experiment 4

Implementation and testing of a bus topology architecture using Ethernet Shields and W5500 Ethernet Modules.
